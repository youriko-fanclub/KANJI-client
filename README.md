## QuickStart
```
$ make init
$ make run
```

## Build on CLI
### Default build
```
$ make build
```

### Debug build
```
$ make build/debug
```

### Release Build
```
$ make build/release
```


## Coding Conventions
- case
  - 変数: snake_case
  - 関数: camelCase
  - class: PascalCase
  - namespace: smallcase
  - enum: PascalCase
  - macro: CAPITAL_SNAKE
- prefix
  - class member field: m_
  - class static member field: s_
- indent
  - default: 4
  - namespace: 0
